# 노란우산공제 상담 가이드 (iM뱅크 영업점 전용)

iM뱅크 창구 직원이 노란우산공제 관련 고객 상담 시 약관·법령 근거가 명확한 안내를 빠르게 진행할 수 있도록 돕는 내부 업무지원 도구입니다.

## 주요 기능

- **상담 시뮬레이터** (세일즈 코치 모드 토글 지원)
- **세일즈 보조 계산기** 3종 (소득공제 절세효과 / 해약환급금 시나리오 / 상품 비교)
- **업무별 가이드** 19개 (가입·납입·소득공제·공제금·해약·대출·수급권보호·구비서류)
- **FAQ** 56개 (약관·시행령·시행규칙 + 노란우산공제 공식 사이트 기반)
- **구비서류 체크리스트** 11개 사유별
- **고객 안내 스크립트 자동 생성**

## 데이터 출처

- 소기업·소상공인공제 운용요강 (2026. 1. 1 시행)
- 조세특례제한법 및 시행령·시행규칙 (2026. 2. 27 시행)
- 중소기업협동조합법 (제121조의2 소멸시효 2026. 6. 3 시행)

## 로컬 실행

\`\`\`bash
npm install
npm run dev
\`\`\`

## 빌드 및 배포

\`\`\`bash
npm run build
\`\`\`

Vercel·Netlify 등 정적 호스팅에 \`dist/\` 폴더를 배포합니다.

## 주의사항

- 본 도구는 직원 안내 보조용으로, 공제금·해약환급금·세금·대출한도 등은 단정 안내를 피하고 중앙회 시스템 조회 결과로 안내해야 합니다.
- 기준이율·부가지급률은 매 분기 변동되며, 법령 개정 시 콘텐츠 업데이트가 필요합니다.
- 고객 개인정보는 입력 금지 (모든 시뮬레이션은 가상 변수로만 진행).
